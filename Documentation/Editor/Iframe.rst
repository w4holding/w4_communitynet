=========================
Iframe Content Element
=========================

Iframe content elements renders a responsive iframe with different aspect ratios. Follow these steps to add a iframe content element to a page

#. Go to module :guilabel:`Web > Page` and select the page where you want to add the iframe content element

#. Add a new content element and select the entry 
   :guilabel:`CommunityNet > Iframe`.

#. Switch to the tab :guilabel:`Configuration` where you can define the content settings. 

#. Fill out all desired fields and click :guilabel:`Save`.
