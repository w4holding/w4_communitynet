=====================================
Container (columns) content elements
=====================================


The Community Net Extension is shipped with the following Container content elements

* Two columns container
* Two columns container with side column
* Three columns container
* Four columns container


Follow these steps to add a container content element to a page.

#. Go to module :guilabel:`Web > Page` and select the page where you want to add the container content element.

#. Add a new content element and select the entry under the 
   :guilabel:`Container`. This will create a new container on the page

   .. image:: ../Images/Container_content_step_1.png
      :class: with-border with-shadow

#.  Add content to each column of the container by clicking on the :guilabel:`+ content` button.

   .. image:: ../Images/Container_content_step_2.png
      :class: with-border with-shadow
