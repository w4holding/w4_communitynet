# TYPO3 Extension `w4_communitynet`

CommunityNet extension is a theme for TYPO3 CMS based on the Bootstrap CSS Framework. CommunityNet comes with a fully configurable Frontend via backend settings and TypoScript.  

## Installation

### Quick guide

- Add repository path to your application composer.json file:

        "repositories": [
            {
                "type": "github",
                "url": "git@github.com:w4holding/w4_communitynet.git"
            }
        ],

- Install via composer:  

        composer require w4services/w4_communitynet
